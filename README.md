# Superstore Sales Dashboard

## Overview
Interactive 3-page sales dashboard built in Power BI.

## Pages
- Sales Overview — KPIs, regional and category analysis
- Profit Analysis — Profit trends, losses, scatter analysis
- Regional Analysis — Matrix table, shipping analysis

## Tools Used
- Power BI Desktop
- DAX Measures
- Data Source: Sample Superstore Dataset

## Key Insights
- Total Sales: $2.30M
- Profit Margin: 12.47%
- West region leads with $725K in sales
- Technology is the top performing category
- Standard class is the most used shipping method
